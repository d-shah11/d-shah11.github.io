Big Data Analyzing

This is my first academic project in my graduate coursework. To analyze insurance risk associated with the truck fleet based on geographic location and time using Hortonworks Data Platform tools and techniques. By completing this project, I learnt the concepts of Hadoop DIstributed File System, to write queries on Hive, important difference between MapReduce and Hadoop, to work on Cloudera VMware Workstation and Hue GUI and finally to integrate the data between HDFS and Tableau.
